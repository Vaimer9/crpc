# Command Line Rich Presence (CRPC)
A Blazingly Fast Rich Presence Command Line Application made in rust

### Current Status: *In Progress*

### JSON Example
```
{
	id: "",
	status: "",
	details: "",
	large: false,
	smalle: false,
	want_buttons: false,
	large_image: "",
	small_image: "",
	large_tool: "",
	small_tool: "",
	buttons: {
		name: "",
		link: "",
		name2: "",
		link2: ""
	}
}
```
I have tried my best to keep the JSON as simple as possible.
